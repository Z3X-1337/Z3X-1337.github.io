Defensive tools with explicit evidence and maturity boundaries.

The portfolio contains one integrated SOC workflow and three focused utilities. Every repository is an installable v0.1.0 CLI with tests, CI installation checks, MIT licensing, a changelog, and a documented roadmap.

01
Primary projectIntegrated SOC workflow

SOC CaseForge

Turns sanitized analyst evidence into a structured local case with a timeline, indicators, explainable findings, and reviewable reports.

Evidence today

  • Installable soc-caseforge CLI
  • SQLite-backed cases and normalized events
  • OpenSSH ingestion, IOC extraction, and deterministic findings
  • Committed demo report and 24 tests across Python 3.10–3.12

Next engineering stage

  • ISO-8601 timestamp and timezone normalization
  • CSV and generic JSON event adapters

  • Evidence manifests, suppressions, and false-positive datasets
