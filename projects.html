Auth Log Analyzer

Analyzes supported OpenSSH authentication logs and reports suspicious source behavior using configurable failure and password-spray thresholds.

Evidence today

  • IPv4 and IPv6 source validation
  • Multiple SSH authentication methods
  • JSON and Markdown reports
  • ATT&CK rationale and 12 tests

Next engineering stage

  • Time-window detections
  • Additional OpenSSH event types
  • CSV export and installable packaging
  • False-positive test datasets
